Ambassadeur de créateurs

Projet client réalisé pour la société « Ambassadeurs de créateurs »

Objectif : créer une plateforme qui met en relation des créateurs et des hôtes susceptibles de vendre leurs créations. Une carte interactive doit être intégrée afin de visualiser les prochaines ventes. Contrainte : un créateur ne peut contacter un hôte que s'il a souscrit à un abonnement.

Technologies utilisées : PHP 7.2 / Symfony 4.4.3 / Doctrine 2 / Twig / HTML 5 / CSS-SCSS / Bootstrap 4 / Javascript / MySQL

Développement de fonctionnalités : gestion des utilisateurs, carte interactive, filtrage des données et pagination, administration des données, etc.

Méthodologie Agile Scrum sur 8 semaines en équipe de 5.

Versionning : Git / Github

Le Kéodex - 2e prix du jury

Projet réalisé dans le cadre d'un hackathon de 48h, en partenariat avec Kéolis

Objectif : créer une application qui redynamise et rajeunit les applications actuelles du réseau TAO en mettant en avant le développement durable.

Proposition : nous avons choisi de mettre en place une application sous forme de gamification. L'utilisateur recherche sur l'ensemble du réseau urbain des objets virtuels à collectionner. Certains objets permettent l'obtention de promotions. Le but étant d'inciter les utilisateurs à prendre les transports en commun.

Technologies utilisées : PHP 7.4 / Symfony 4.4 / Doctrine 2 / Twig / HTML 5 / CSS-SCSS / Bootstrap 4 / Javascript

Développement de fonctionnalités : carte interactive (librairie Leaflet) avec affichage dynamique de données, intégration d'un scan de QRcode, utilisation de l'API de la TAO, objets virtuels récoltés sous forme de collection, triés par ligne du réseau TAO.

Hackathon de 48h en équipe de 4.

Versionning : Git / Github

  • Date : décembre 2019
  • Client : Kéolis
